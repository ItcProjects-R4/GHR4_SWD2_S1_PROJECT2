# Student Grade System

A simple, interactive web application built with **HTML**, **CSS**, and **JavaScript**. It converts academic scores (0-100) into letter grades (A, B, C, etc.) with a clean, smooth user interface.

## Built With

- **HTML5**
- **CSS3**
- **Vanilla JavaScript**
- **FontAwesome 6**
- **Google Fonts**


## Grading Scale

| Score Range | Letter Grade | Status    |
| ----------- | ------------ | --------- |
| 90 - 100    | **A+**       | Excellent |
| 85 - 89     | **A**        | Excellent |
| 80 - 84     | **B+**       | Very Good |
| 75 - 79     | **B**        | Very Good |
| 70 - 74     | **C+**       | Good      |
| 65 - 69     | **C**        | Good      |
| 60 - 64     | **D**        | Pass      |
| 0 - 59      | **F**        | Fail      |
